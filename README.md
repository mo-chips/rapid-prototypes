## Why this repository?

I keep my experimental work separate from my primary engineering portfolio.

Portfolio repositories focus on:

- Backend architecture
- Networking
- Software engineering principles
- Maintainable code

This repository focuses on:
- Rapid prototyping
- UI ideas
- AI workflows
- New technologies
- Fun experiments
